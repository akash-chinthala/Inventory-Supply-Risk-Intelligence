📦 Inventory & Supplier Risk Analysis
📌 Project Overview
This project analyzes inventory levels and supplier performance to identify risks in the supply chain. It focuses on detecting stock shortages, delivery delays, and high-risk suppliers to help businesses take proactive decisions.

🎯 Objectives
* Monitor stock levels across products
* Identify delayed deliveries
* Detect high-risk suppliers
* Improve supply chain efficiency

🧩 Dataset
The dataset includes:
* product_id → Product identifier
* product_name → Product name
* supplier_id → Supplier identifier
* supplier_name → Supplier name
* stock_quantity → Available stock
* order_quantity → Ordered quantity
* order_date → Order placed date
* expected_delivery → Planned delivery date
* actual_delivery → Actual delivery date

🛠 Tools & Technologies
* Excel → Data cleaning (nulls, duplicates, formatting)
* SQL → KPI calculations & risk segmentation
* Power BI → Dashboard creation

🔍 Key Features
✔ Row-Level Analysis
* Calculated delay_days using delivery dates
* Categorized:
    * On-Time
    * Moderate Delay
    * High Delay
* Classified stock levels:
    * Low Stock
    * Medium Stock
    * High Stock
* Created risk segments:
    * Low Risk
    * Medium Risk
    * High Risk

✔ Supplier-Level Analysis
* Total orders per supplier
* Average delay per supplier
* Count of delayed orders
* Supplier risk classification:
    * High Risk Suppliers
    * Medium Risk
    * Low Risk
